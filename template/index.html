This is an API for the English Premier League sourcing information about players and teams.

How to use the API?


Get Data about Players
If you want to get information about specific player go to all_players to find the ID for the player you are intrested in and then with GET method on "eplapi.azurewebsites.net/player/{player_id}" pass the id of the player as parameter.

Get Data about Teams
If you want to get information about specific team go to all_teams to find the ID for the team you are intrested in and then with GET method on "eplapi.azurewebsites.net/team/{team_id}" pass the id of the team as parameter.
All data is set in UTC time zone.

The data has been gathered from www.understat.com.
